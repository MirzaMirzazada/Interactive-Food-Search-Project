# Interactive Food Search Project

## Overview
This project is part of the **Vector Databases for RAG: An Introduction** course. It demonstrates an interactive search application using Python, designed to explore vector databases and retrieval-augmented generation (RAG) techniques.

## Project Structure
- `interactive_search.py` — Main script to run the interactive search application.
- `shared_functions.py` — Contains helper functions used across the project.
- Other supporting files as required by the course.

## Prerequisites
- Python 3.10 (or compatible version available in Cloud IDE)
- No additional external libraries needed unless specified by the course.
